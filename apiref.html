class MMA8452Q

Public Type

MMA8452Q_SYSMOD enum

Values:

  • STANDBY -

    Stand-by system mode

  • WAKE -

    Wake system mode

  • SLEEP -

    Sleep system mode

MMA8452Q_EVENTS enum

Values:

  • FREEFALL_MOTION = 0x01 -

    Freefall/motion event

  • PULSE = 0x02 -

    Pulse (tap) event

  • ORIENTATION = 0x04 -

    Orientation change event

Public Functions

MMA8452Q()

int begin(void)

Initialize the MMA8452Q.

uint8_t status(void)

Read the status of the MMA8452Q.

uint8_t sysmod(void)

Read the system mode of the MMA8452Q.

Example: 

switch (accel.sysmod()) {
  case MMA8452Q::STANDBY:
    Serial.println("StandBy");
    break;

  case MMA8452Q::SLEEP:
    Serial.println("Sleep");
    break;

  case MMA8452Q::WAKE:
    Serial.println("Wake");
    break;
}
Return
One of the MMA8452Q::MMA8452Q_SYSMOD values.

uint8_t intSource(void)

Read the source pin of the last interrupt.

void scale(uint8_t scale)

Set the scale of the MMA8452Q.

Before calling this method, the MMA8452Q must be disabled using the active() method.

Example: 

accel.active(false);
accel.scale(2);
accel.active(true);
Parameters
  • scale -

    scale can be 2, 4, or 8.

void offset(int8_t off_x, int8_t off_y, int8_t off_z)

Calibrate the MMA8452Q.

Before calling this method, the MMA8452Q must be disabled using the active() method.

Example: 

accel.active(false);
accel.offset(5, -6, 10);
accel.active(true);
Parameters
  • off_x -

    X axis calibration.

  • off_y -

    Y axis calibration.

  • off_z -

    Z axis calibration.

void axes(int * axes)

Read the raw values of the axes.

Example: 

int axes[3];
int x, y, z;

accel.axes(axes);

x = axes[0];
y = axes[1];
x = axes[2];
Parameters
  • axes -

    output array of size 3 (must be allocated by the user).

bool orientation(uint8_t * value)

Read the orientation value.

Before calling this method, the orientation detection must be activated using the detectOrientation() method.

Example: 

uint8_t orientation;

if (accel.orientation(&orientation)) {
  ...
}
Return
Whether the orientation has changed since the last read.
Parameters
  • value -

    output value.

int landscape(uint8_t orientation)

Read the landscape orientation status.

Example: 

uint8_t orientation;

accel.orientation(&orientation));

switch (accel.landscape(orientation)) {
  case HIGH:
    Serial.println("Landscape Right");
    break;

  case LOW:
    Serial.println("Landscape Left");
    break;

  default:
    Serial.println("No Orientation");
    break;
}
Return
Whether the landscape orientation is right or left.
Parameters
  • orientation -

    The orientation set by the orientation() method.

int portrait(uint8_t orientation)

Read the portrait orientation status.

Example: 

uint8_t orientation;

accel.orientation(&orientation));

switch (accel.portrait(orientation)) {
  case HIGH:
    Serial.println("Portrait Up");
    break;

  case LOW:
    Serial.println("Portrait Down");
    break;

  default:
    Serial.println("No Portrait");
    break;
}
Return
Whether the portrait orientation is up or down.
Parameters
  • orientation -

    The orientation set by the orientation() method.

int backFront(uint8_t orientation)

Read the back/front orientation status.

Example: 

uint8_t orientation;

accel.orientation(&orientation));

if (accel.backFront(orientation))
  Serial.println("Back");
else
  Serial.println("Front");
Return
Whether the orientation is back or front.
Parameters
  • orientation -

    The orientation set by the orientation() method.

void wakeOn(bool enable, uint8_t events)

Enable/disable auto-wake on specific events.

Before calling this method, the MMA8452Q must be disabled using the active() method.

Example: 

accel.active(false);
accel.wakeOn(true, FREEFALL_MOTION | ORIENTATION);
accel.active(true);
Parameters
  • enable -

    Whether to enable or disable the auto-wake on the selected events.

  • events -

    Bit mask listing the desired events. See MMA8452Q::MMA8452Q_EVENTS.
